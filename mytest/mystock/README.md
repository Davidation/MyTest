# 同花顺量化交易系统

这是一个基于同花顺的小规模量化交易系统，能够根据股市行情和仓位情况，结合科学的算法指标，自动做出买入或卖出操作。

## 系统架构

- **数据获取模块**：从同花顺获取实时行情数据
- **仓位管理模块**：跟踪和管理用户持仓情况
- **技术指标计算模块**：计算MA、MACD、RSI等技术指标
- **交易策略模块**：基于指标信号生成买卖决策
- **交易执行模块**：通过同花顺API执行交易操作
- **监控和日志模块**：监控系统运行状态和记录操作日志

## 技术栈

- Python 3.9+
- 同花顺OpenAPI
- pandas (数据处理)
- numpy (数值计算)
- TA-Lib (技术指标计算)
- logging (日志管理)

## 快速开始

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 配置同花顺API
   - 在 `config.py` 中填写API密钥和相关配置

3. 运行系统
```bash
python main.py
```

## 配置说明

系统配置文件为 `config.py`，包含以下配置项：
- API密钥和账号信息
- 交易策略参数
- 技术指标参数
- 风险控制参数

## 策略说明

系统支持多种交易策略，可在 `strategies/` 目录下添加自定义策略。默认策略基于以下指标：
- 移动平均线(MA)：判断趋势
- MACD：判断动量
- RSI：判断超买超卖

## 风险控制

系统内置风险控制机制：
- 单只股票最大持仓比例
- 每日最大交易次数
- 止损止盈设置
- 异常情况自动暂停

## 日志管理

系统日志存储在 `logs/` 目录下，包括：
- 交易日志：记录所有交易操作
- 策略日志：记录策略信号和决策过程
- 系统日志：记录系统运行状态和异常情况
